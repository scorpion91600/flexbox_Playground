# 🚀 Dev Academy - Ressources Front-End

Une collection complète d'outils et de ressources pour maîtriser le développement front-end moderne. Que vous soyez débutant ou développeur expérimenté, ces outils interactifs et cheat sheets vous aideront à apprendre et à perfectionner vos compétences en HTML, CSS et mise en page.

![Dev Academy Preview](https://via.placeholder.com/800x400/1a1a2e/ffffff?text=Dev+Academy+Preview)

## 📚 Contenu du Projet

### 1. **Cheat Sheets Complètes**
- **[HTML Cheat Sheet](html.html)** : Toutes les balises HTML5 essentielles avec exemples
- **[CSS Cheat Sheet](css.html)** : Propriétés CSS, sélecteurs, animations et plus

### 2. **Outils Interactifs**
- **[Flexbox Academy](main2.html)** : Outil interactif pour maîtriser Flexbox
- **[CSS Text Styler](texte.html)** : Prévisualisation en temps réel des propriétés textuelles

## 🎯 Fonctionnalités

### Cheat Sheets
✅ Références rapides pour HTML et CSS
✅ Exemples de code clairs et commentés
✅ Recherche facile par catégories
✅ Design responsive et moderne

### Outils Interactifs
🔧 **Flexbox Academy** :
- Visualisation en temps réel des propriétés Flexbox
- Contrôles interactifs pour chaque propriété
- Exemples concrets avec code généré
- Mode "défi" pour tester vos connaissances

🔧 **CSS Text Styler** :
- Prévisualisation instantanée de 15+ propriétés textuelles
- Génération automatique du code CSS
- Exemples de texte adaptés à chaque propriété
- Réinitialisation facile des paramètres
